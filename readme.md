# Charla para la *Semana de la Física UNSJ 2026*

**Titulo:** Sistemas Complejos y Redes Complejas
**Autor:** Dr. Juan I. Perotti
**Afiliaciones:** Instituto de Física Enrique Gaviola (IFEG-CONICET), Facultad de Matemática, Astronomía, Física y Computación (FAMAF-UNC)

**Resumen:**

Muchos sistemas naturales, sociales y tecnológicos están formados por una gran cantidad de elementos que interactúan de manera no trivial, dando lugar a comportamientos colectivos emergentes difíciles de anticipar a partir de sus componentes individuales. En esta charla introductoria se presentarán las ideas fundamentales de la ciencia de sistemas complejos y cómo la física estadística proporciona herramientas para comprender sistemas tan diversos como el cerebro, las redes sociales o la economía. En particular, discutiremos el papel central que desempeñan las redes complejas en la descripción de las interacciones, la organización y la dinámica colectiva de este tipo de sistemas.
